# Student-Performance-Ml-pj

Machine Learning project to predict student academic performance using Python and Streamlit

Student Performance ML Project



This project predicts student academic performance using Machine Learning

and deploys the model using Streamlit.



Features

\- Data preprocessing

\- Machine learning model

\- Model evaluation

\- Streamlit web app



Project Structure



data/ → dataset

notebooks/ → exploration

src/ → training scripts

models/ → saved models

app/ → streamlit app



Installation



pip install -r requirements.txt



Run App



streamlit run app/app.py
